   Clientwork: Security Assessment Simulation

This project is a multi-stage web application designed to simulate a security assessment environment. It demonstrates front-end transitions, user interface design for "Secure Administrator Access" portals, and integration of multimedia elements.

Features

    Multi-Stage Workflow: A logical progression from a simulated social media account alert to a secure terminal login.

    Responsive Professional UI: Designed with a focus on enterprise-grade aesthetics using modern CSS.

    Automatic Fullscreen Mode: Implementation of the Fullscreen API to create an immersive, high-stakes simulation environment.

    Multimedia Integration: Supports .mp4 voice messages and audio cues triggered by user interactions.



    File Structure


    File,Description
index.html,The main structure of the simulation.
style.css,Professional styling and responsive layout rules.
script.js,Logic for stage transitions and Fullscreen API calls.
client.mp4,Integrated voice/audio assets.




How to Run

    Clone the repository from your GitHub account: Spectrax011/clientwork.

    Ensure you have a local server running (e.g., XAMPP or VS Code Live Server).

    Open index.html in your browser.

    Interact with the Account Security Alert to enter the full-screen simulation.
